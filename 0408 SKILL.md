# 快速掌握主题 - Rapid Learning

一个系统化的快速学习框架，结合认知科学原理与AI辅助，帮助用户高效掌握任何新主题。

## 核心学习原则

1. **主动回忆 > 被动阅读** - 通过自测强化记忆
2. **费曼技巧** - 用简单语言教授他人来检验理解
3. **间隔重复** - 科学安排复习时机
4. **项目驱动** - 在实战中建构知识
5. **渐进披露** - 按需加载信息，避免认知过载

## 使用流程

### 阶段1：学习诊断（5分钟）

启动任何学习前，先了解：

- **目标主题**：具体要学什么？
- **当前水平**：0-10分自评
- **学习目的**：实用/应试/兴趣/工作/研究？
- **时间约束**：可用总时长？每日可投入时间？
- **输出目标**：学成后要做什么？（项目/考试/教学/应用）

### 阶段2：路径规划

根据主题类型选择学习模式：

| 主题类型 | 推荐模式 | 核心策略 |
|---------|---------|---------|
| 技术/编程 | 项目驱动 | 脚手架项目 → 概念深挖 → 刻意练习 |
| 理论/学术 | 概念建构 | 知识图谱 → 费曼讲解 → 主动回忆 |
| 操作技能 | 模仿练习 | 观察 → 分解练习 → 完整执行 |
| 语言学习 | 沉浸输入 | 输入轰炸 → 输出强制 → 间隔重复 |
| 考试备考 | 靶向突破 | 考纲分析 → 弱点诊断 → 模拟测试 |

### 阶段3：执行学习循环

每个学习单元遵循 **ICAP模型**：

```
Input（输入）→ Construct（建构）→ Apply（应用）→ Polish（精炼）
```

**Input（输入）- 10分钟**
- 提供核心概念的多模态解释（文字+图表+代码/示例）
- 使用 `create_mindmap.py` 生成概念图
- 遵循 "先整体后局部" 原则

**Construct（建构）- 15分钟**
- 用户进行费曼输出：用自己的话解释概念
- AI扮演"好奇的初学者"进行追问
- 识别并澄清理解盲区

**Apply（应用）- 20分钟**
- 完成实战练习或迷你项目
- 解决真实问题
- 调用 `assess_mastery.py` 检查理解

**Polish（精炼）- 5分钟**
- 生成Anki卡片用于间隔复习
- 记录学习笔记和顿悟时刻
- 规划下次复习时间

### 阶段4：掌握度评估

使用掌握度量表进行阶段性检查，详见 [references/mastery-rubric.md](references/mastery-rubric.md)

**四级掌握标准：**
- L1 认知：能识别和描述概念
- L2 理解：能解释原理和关联
- L3 应用：能独立解决问题
- L4 精通：能教授他人并创新

### 阶段5：巩固与扩展

- 生成复习计划（1天后、3天后、7天后、30天）
- 推荐进阶资源和相邻主题
- 设计"教授他人"的输出形式

## 内置工具使用

### 生成Anki卡片
```bash
python scripts/generate_anki.py --topic "主题名" --level L2 --output deck.apkg
```

### 生成思维导图
```bash
python scripts/create_mindmap.py --input concepts.md --output mindmap.mmd
```

### 掌握度自测
```bash
python scripts/assess_mastery.py --topic "主题名" --level 目标等级
```

## 学习效果优化建议

1. **番茄工作法**：25分钟专注 + 5分钟休息
2. **多样化练习**：在不同场景下应用知识
3. **交叉学习**：交替学习相关主题，增强辨别力
4. **睡眠固化**：学习后保证充足睡眠
5. **即时反馈**：尽快验证理解是否正确

## 常见学习误区警示

❌ 被动重读笔记 → ✅ 主动闭卷回忆
❌ 划线高亮 → ✅ 用自己的话总结
❌ 题海战术 → ✅ 针对性弱点突破
❌ 单次长时间 → ✅ 分散多次学习
❌ 回避困难 → ✅ 刻意练习难点
