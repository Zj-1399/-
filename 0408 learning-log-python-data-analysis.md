#### 第1课：数据计算器
**时间**：2026-04-08

**核心知识点**：
- pandas 读取CSV: `pd.read_csv()`
- 查看数据: `df.head()`, `df.describe()`
- 选择列: `df['列名']`
- 统计计算: `.mean()`, `.max()`, `.min()`, `.sum()`

**项目作业**：学生成绩分析器
```python
import pandas as pd

# 读取数据
df = pd.read_csv('scores.csv')

# 计算总分
df['总分'] = df['语文'] + df['数学'] + df['英语']
df['平均分'] = df['总分'] / 3

# 找出最高分
max_chinese = df['语文'].max()
max_student = df[df['语文'] == max_chinese]['姓名'].values[0]

# 输出报告
print("=== 成绩分析报告 ===")
print(f"语文最高分: {max_chinese} ({max_student})")
print(f"全班平均总分: {df['总分'].mean():.2f}")
print("\n成绩排名:")
print(df.sort_values('总分', ascending=False))
```
