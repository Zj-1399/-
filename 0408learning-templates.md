# 学习模板库

针对不同学习场景的即用模板。

---

## 从零开始模板

适用于：全新领域入门

### 第1轮：建立认知地图（30分钟）
1. **全景概览**：主题是什么？解决什么问题？
2. **核心概念**：3-5个最基础的概念
3. **知识图谱**：概念间的关系图
4. **学习路线**：从入门到精通的里程碑

### 第2轮：首个里程碑（2-4小时）
1. **最小可运行示例**：做出第一个能工作的成果
2. **概念关联**：理解示例背后的原理
3. **修改实验**：尝试修改参数看效果
4. **记录问题**：遇到的不理解的点

### 第3轮：系统学习（按需）
1. **按图索骥**：根据知识图谱逐个攻克
2. **项目驱动**：每个概念配合小练习
3. **费曼输出**：向AI解释每个学到的概念
4. **生成卡片**：制作复习卡片

### 第4轮：整合应用（2-4小时）
1. **综合项目**：结合多个概念完成项目
2. **代码/笔记回顾**：整理学习笔记
3. **教学输出**：写一篇入门教程

## 考前冲刺模板

适用于：有明确考试目标，时间紧迫

### 阶段1：考纲分析（20分钟）
1. **分值分布**：各章节权重
2. **题型分析**：选择/简答/计算/论述
3. **难度分级**：基础/中等/困难占比
4. **时间规划**：倒排复习时间表

### 阶段2：知识梳理（40%时间）
1. **快速扫盲**：通读教材/讲义，建立框架
2. **重点标注**：高频考点、易错点
3. **公式/概念卡片**：制作速记卡
4. **思维导图**：每章一张核心图

### 阶段3：题型突破（40%时间）
1. **分类刷题**：按题型和知识点分类
2. **错题本**：记录错误原因和正确思路
3. **时间训练**：模拟考试时限
4. **套路总结**：常见解题模式

### 阶段4：模拟冲刺（20%时间）
1. **全真模拟**：完整模拟考试
2. **弱点补强**：针对薄弱环节
3. **状态调整**：生物钟、心态
4. **考前速记**：最后再看一遍核心点

## 深度研习模板

适用于：已入门，希望深入理解

### 阶段1：广度扩展
1. **经典论文/著作**：阅读领域经典
2. **不同视角**：对比不同教材/讲师的讲解
3. **历史脉络**：了解概念的发展过程
4. **相邻领域**：学习与主题相关的其他知识

### 阶段2：深度挖掘
1. **第一性原理**：追问"为什么"直到基础公理
2. **数学推导**：理解公式背后的推导过程
3. **边界条件**：了解方法的适用范围和局限
4. **反例分析**：研究什么情况下会失败

### 阶段3：批判思考
1. **优缺点分析**：方法的trade-off
2. **替代方案**：了解其他解决方法
3. **前沿进展**：关注最新研究和趋势
4. **个人观点**：形成自己的判断和偏好

### 阶段4：知识输出
1. **技术博客**：撰写深度文章
2. **演讲分享**：准备分享材料
3. **教学相长**：指导他人学习
4. **实践创新**：尝试改进或创新

## 技能练习模板

适用于：需要身体/操作协调的技能

### 观察阶段
1. **完整观摩**：观看专家示范，不中断
2. **分解记录**：将动作分解为步骤
3. **关键点标记**：识别难点和要点
4. **常见问题**：了解典型错误

### 模仿阶段
1. **慢速模仿**：放慢速度，保证正确
2. **即时反馈**：有人/录像检查动作
3. **分段练习**：先练A部分，再练B部分
4. **重复巩固**：正确动作重复10次以上

### 整合阶段
1. **连贯执行**：将分段整合为完整流程
2. **速度提升**：逐渐加快到正常速度
3. **变化适应**：在不同条件下练习
4. **压力测试**：在有时间压力/干扰下练习

### 自动化阶段
1. **闭眼执行**：不看也能正确完成
2. **边做边讲**：能同时口述步骤
3. **传授他人**：指导初学者
4. **持续精进**：寻找微小改进空间
