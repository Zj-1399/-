# Learn Topic - 快速掌握任意主题

## 概述

这是一个基于连接主义、建构主义、认知负荷理论和主动回忆等学习理论的学习助手。它帮助你快速建立对某个主题的系统性理解，并通过主动回忆和间隔复习巩固知识。

## 核心学习流程

### 第一步：探索主题结构 `/learn explore <主题>`
- **目的**：建立知识网络地图（Connectivism）
- **输出**：
  1. 主题的核心概念列表（5-10个）
  2. 概念之间的关系图谱
  3. 推荐学习顺序（按依赖排序）
- **格式**：使用 Markdown 列表和关系图

### 第二步：渐进学习 `/learn study <主题> [--depth=1-3]`
- **目的**：控制认知负荷（Cognitive Load Theory）
- **depth=1**：入门级 - 用生活类比解释核心概念
- **depth=2**：标准级 - 包含关键原理和示例
- **depth=3**：深入级 - 包含技术细节和边界情况
- **每次只呈现一个核心概念**，避免信息过载

### 第三步：主动回忆 `/learn quiz <主题> [--count=3-10]`
- **目的**：通过测试强化记忆（Active Recall）
- **题型**：概念解释、关系识别、应用场景
- **每次只问1-3个问题**，回答后揭示答案

### 第四步：追踪进度 `/learn track <主题>`
- **目的**：记录学习状态，支持间隔复习（Spaced Repetition）
- **显示**：
  - 已学习概念数量
  - 下次复习时间
  - 薄弱环节

### 第五步：教学相长 `/learn teach <主题>`
- **目的**：暴露知识盲点（Feynman Technique）
- **方式**：让你用自己的话解释概念，我来指出不清除的地方

## 命令速查

| 命令 | 功能 |
|------|------|
| `/learn explore <主题>` | 探索主题结构 |
| `/learn study <主题>` | 开始学习 |
| `/learn quiz <主题>` | 测验 |
| `/learn track` | 查看所有主题进度 |
| `/learn track <主题>` | 查看特定主题进度 |
| `/learn teach <主题>` | 教学相长 |
| `/learn review` | 复习所有到期主题 |
| `/learn reset <主题>` | 重置学习进度 |

## 设计原则

1. **渐进性**：从简单到复杂，每次只学一个概念
2. **主动性**：通过问题引导思考，而非被动阅读
3. **连接性**：始终强调新知识与已有知识的联系
4. **反思性**：定期让你反思学习过程
5. **实用性**：每学一个概念都给出应用场景

## 示例对话

```
你：/learn quiz machine-learning --count=3

1. 监督学习和无监督学习的主要区别是什么？

AI：[评估回答，给出反馈]
```
